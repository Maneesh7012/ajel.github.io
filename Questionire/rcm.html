RCM Assessment

Excellence

RCM used proactively for new projects. RCM, maintenance planning, and support analysis used before new equipment / systems put into service.

Competence

PM program fully developed using RCM / PMO, improved using RCFA. RCM results evident in procedure changes, training, & equip mods.

Understanding

RCM program in use for critical equipment. PM program blends OEM recommendations with experience & RCM results.

Awareness

Downtime analysis is performed & some improvements are implemented. PM program is being followed.

Innocence

Plenty of downtime, but no analysis of causes or attempts to improve. PM program missing or not followed. Production complains about how badly maintenance manages assets.
